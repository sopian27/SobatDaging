Description

Cloudify is a HTML One Page Template for for Web Hosting Companies. The one page Template has all the needs for a Hosting Company. Domain search ,registration, live chat and many more features. Modern and clean pricing tables that let you display your hosting prices for different server configurations and hosting packages aswell as functional contact & subscription forms.

Enjoy!

Change Logo

Its very easy to change your logo.
All you have to do is open your desired html file on a code editor and change the source/name of the logo to your logo.

Any-file.html 
                                            
<!-- Logo -->
<div class="col-md-2 col-sm-3 col-xs-3">
    <a class="navbar-brand" href="index.html"><img src="images/logo.svg" alt="logo"></a><!-- INSERT YOUR LOGO HERE -->
</div>

Fullscreen Image Background

To change the background image for the "fullscreen-image.html" page you have to change the background image link to the image that you desire to place as a background on file style.css

css/style.css 
                                            
.demo2 {
    height: 100%;
    height: -webkit-calc(100% - 47px);
    height: -moz-calc(100% - 47px);
    height: calc(100% - 47px);
    background-size: cover;
    background: url('../images/img/image1.jpg'); //  <=====   Change this link
}

Fullscreen Video Background


To change the Video background for the slider or the "fullscreen-video.html" you have to:


1. Go to the folder video:



2. You will find the videos in 3 formats. .mp4 .ogg and .webm and a JPG image. You can convert your videos in this 3 format very easily via http://video.online-convert.com/ and replace them with your new converted videos.



Note: If your filename is not "video" and has a dfferent name you can change it easily by changing the source and linking it to your video, on the page that you want (index.html, onepage.html, fullscreen-video.html etc).

onepage.html 
                                            
<video class="video-bg " preload="auto" loop="" autoplay="" poster="video/video.jpg">
    <source src="video/video.mp4" type="video/mp4">
    <source src="video/video.webm" type="video/webm">
    <source src="video/video.ogv" type="video/ogg">
</video>

Animated Clouds

To delete the animated clouds from the section you have to delete this part of code:

onepage.html 
                                            
<!-- Clouds that are used in the animation -->
<div class="clouds">
    <img src="images/clouds/cloud1.svg" alt="" class="cloud1">
    <img src="images/clouds/cloud2.svg" alt="" class="cloud2">
    <img src="images/clouds/cloud3.svg" alt="" class="cloud3">
    <img src="images/clouds/cloud1.svg" alt="" class="cloud4">
    <img src="images/clouds/cloud3.svg" alt="" class="cloud5">
</div>


OR just add this code on style.css:

css/style.css 
                                            
/* Hide all Animated Clouds */
.clouds{display: none;}

/* For certain sections you could use this:
just add your "#section-id" after section */
section .clouds{display: none;}

Contact Form

Contact form will use php to send emails. So just open email.php file that can be found on the php folder. Change the current email "youremail@cloudify.com" to your email address and you are ready to go.


NOTE: Contact form require web server to send emails. Emails can go to your spam/junk folder.

php/email.php 
                                            
########################################################################
########################################################################

$site_title     = "Cloudify"; //Site Title
$to_email       = "youremail@cloudify.com"; //Replace with your email address
$subject        = "Cloudify - Email "; //Replace with your email default subject

Subscription

Default: Mailchimp

Current active subscribe form will save emails to mail chimp list.


To connect your mail chimp list, open custom.js that can be found in the JS folder. Find the following script and place your mailchimp list url into the "url" as shown on the image below.

js/custom.js 
                                            
/*----------------------------------------------------
  MAILCHIMP
----------------------------------------------------*/
$('.mailchimp').ajaxChimp({
    callback: mailchimpFunct,
    url: "your-mailchimp-url-here" //Replace this with your own mailchimp post URL. Paste the url inside "".  
});

Option 2: Save Email in a Text file

This option currently disabled. To enable it all you have to do is go to the subscription form and delete the "-not" from the id.

Disabled:

index.html 
                                            
                                    
<!-- Start of Form -->
<form action="#" class="form-inline mailchimp" id="subscribtion-form-not"> <!-- <=== delete this "-not" -->
    <!-- subscribe result -->
    <div id="subscribe-result"></div>
    <!-- end of subscribe result -->

Enabled:

index.html 
                                            
                                    
<!-- Start of Form -->
<form action="#" class="form-inline mailchimp" id="subscribtion-form">
    <!-- subscribe result -->
    <div id="subscribe-result"></div>
    <!-- end of subscribe result -->


This Subscription option will save all emails in email-list.txt that can be found on the folder "php/subscribe".
